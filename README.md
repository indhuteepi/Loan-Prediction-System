💳 Loan Prediction System

A Machine Learning web app built with Logistic Regression and Streamlit to predict whether a loan application will be approved or rejected based on user inputs.

📌 Overview

This project uses applicant details like income, credit history, education, and employment status to automate loan approval prediction.

🚀 Features
Data preprocessing & cleaning
Exploratory Data Analysis (EDA)
Logistic Regression model
Interactive Streamlit UI
Real-time predictions
Probability score output
🛠️ Tech Stack
Python
Pandas, NumPy
Matplotlib, Seaborn
Scikit-learn
Streamlit
📂 Dataset

Kaggle Loan Prediction Dataset:
https://www.kaggle.com/datasets/altruistdelhite04/loan-prediction-problem-dataset
